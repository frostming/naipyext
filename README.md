# naipyext

Nasy IPython Extensions.

Add

    c.InteractiveShellApp.extensions = [
        "autoreload",
        "naipyext.autotime",
        "naipyext.be",
        "naipyext.ns"
    ]

to ipython config file `$HOME/.ipython/profile_default/ipython_config.py`
